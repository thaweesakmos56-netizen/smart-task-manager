# smart-task-manager

ระบบจัดการงาน (Task Management System) สำหรับช่วยติดตามและจัดการงานในแต่ละวัน สามารถเพิ่ม แก้ไข ลบ และดูรายการงานได้อย่างง่ายดาย เหมาะสำหรับใช้บริหารงานส่วนตัวหรือใช้เป็นตัวอย่างโปรเจคสำหรับนักพัฒนาซอฟต์แวร์

---

## 📌 คุณสมบัติของระบบ

- เพิ่มรายการงานใหม่
- แก้ไขรายละเอียดงาน
- ลบรายการงาน
- แสดงรายการงานทั้งหมด
- จัดการสถานะของงาน (เช่น Pending / Completed)
- อินเตอร์เฟซใช้งานง่าย

## 📂 โครงสร้างโปรเจค


smart-task-manager/
│
├── index.php # หน้าแสดงรายการงาน
├── add-task.php # เพิ่มงานใหม่
├── edit-task.php # แก้ไขงาน
├── delete-task.php # ลบงาน
├── config.php # ตั้งค่าการเชื่อมต่อฐานข้อมูล
├── style.css # ไฟล์ตกแต่งหน้าเว็บ
└── README.md # เอกสารโปรเจค

## ⚙️ วิธีติดตั้งและใช้งาน

1. Clone โปรเจคจาก GitHub


git clone https://github.com/thaweesakmos56-netizen/smart-task-manager.git


2. นำโปรเจคไปไว้ในโฟลเดอร์


xampp/htdocs/


3. เปิด XAMPP แล้ว Start

- Apache
- MySQL

4. สร้างฐานข้อมูลใน phpMyAdmin


Database name: smart_task_manager


5. Import ไฟล์ฐานข้อมูล (ถ้ามี)

6. เปิดเว็บใน Browser


http://localhost/smart-task-manager
